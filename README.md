# Four-Up
An implementation of the game Four Up in Assembly.

[Four Up](https://waymoot.org/home/fourup/board.html)

# Requirements
- Set Up  [MASM]
- Set up  [DOSBOX] 

# Compile
```sh
 ml fourup.asm
 fourup.exe
```


  [MASM]: <https://sourceforge.net/projects/masm611/>
  [DOSBOX]: <https://www.dosbox.com/>
